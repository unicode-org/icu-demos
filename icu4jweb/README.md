### ICU Java Web Demos

# BUILDING

- `mvn package`

# RUNNING WITH [docker](https://docker.io)

- `sh build-docker.sh`
- `docker run --rm -p 8083:8080 unicode/icu4jweb:latest`  (^C to stop and delete the container)
- now go to [http://127.0.0.1:8083/icu4jweb/](http://127.0.0.1:8083/icu4jweb/)
